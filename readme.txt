Create an Open Search Document for your blog.

== Installation ==

1. Upload `osd.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the *Plugins* menu in WordPress
3. Replace your `<header>` tags by `<head profile="<?php head_profile(); ?>` in your templates
4. Reload the permalinks trough the *Options > Permalinks* menu
